# Terminal Art

ASCII art gallery with cute animals — terminal and web versions.

## Tech Stack
- Python + Textual framework (terminal UI)
- Vanilla HTML/JS (web version)

## Key Files
- `app.py` — Python terminal app
- `generators/*.py` — individual art modules
- `index.html`, `main.js` — web version
- `requirements.txt` — Python deps

## Rules
- Do not push to main without /deploy
